Python Task Automation

This project demonstrates simple task automation using Python scripts.
It combines multiple small real-life automation tasks into a single program.

Features

1. Move all ".jpg" files from one folder to another
2. Extract email addresses from a ".txt" file
3. Scrape the title of a webpage and save it

Technologies Used

- Python
- os module
- shutil module
- re (Regular Expressions)
- requests
- BeautifulSoup

Project Files

task_automation.py – Main Python script
input.txt – Sample text file containing email addresses
requirements.txt – Required Python libraries

Installation

Install required libraries:

pip install -r requirements.txt

How to Run

Run the Python script:

python task_automation.py

Then choose an option:

1 → Move JPG files
2 → Extract emails
3 → Scrape website title

Output

- JPG files will move to destination folder
- Extracted emails will be saved in "extracted_emails.txt"
- Webpage title will be saved in "webpage_title.txt"
